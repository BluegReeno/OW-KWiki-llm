# Offshore Wind Knowledge Bundle

A portable, continuously-updated knowledge base on the global offshore
wind industry — developers, projects, technology, and policy — in the
**Open Knowledge Format (OKF)**: plain markdown any AI agent can read.

## How to use it

Paste this to your AI coding assistant (Claude Code, Cursor, Codex, Gemini CLI, …):

```
Here's a knowledge bundle: <this repo/folder>
Read its README and set it up so I can search over it. Then tell me what's inside.
```

## Agent setup instructions

You're being asked to make this knowledge bundle available so the user
can ask questions about the offshore wind sector. Follow these steps.

### 1. What this is

A folder of markdown files, each with a small YAML frontmatter block,
navigated via `index.md`, following the
[Open Knowledge Format spec](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md).
Read it **directly**: no database, no embeddings, no API. It ships a
tiny, dependency-free CLI — `okf-cli.py` (Python standard library only)
— for fast navigation and search.

### 2. Use the CLI

- `python okf-cli.py index [subpath]` - print an index (start at the root; e.g. `python okf-cli.py index companies`)
- `python okf-cli.py find "<query>"` - ranked keyword search across the bundle
- `python okf-cli.py read <path>` - print a page, e.g. `python okf-cli.py read projects/dogger-bank`

(You can also just open the markdown files directly - `okf-cli.py` is only a convenience.)

### 3. How to answer the user's questions

1. `python okf-cli.py find "<the user's topic>"` (or read `index.md`) to locate the relevant pages.
2. `python okf-cli.py read` the specific pages - only what's relevant, not the whole bundle (progressive disclosure).
3. Follow the relative markdown links between concepts (e.g. a project links to its developer companies and the technologies it uses).
4. Answer grounded in those pages, and **cite the source** (`resource` field in the frontmatter, or the `# Citations` section) when available.
5. Check `digests/index.md` for the most recent industry news if the question is about something recent — this bundle updates continuously.

This is read-only reference knowledge for a Q&A session - don't modify the bundle unless explicitly asked to curate it.

## What's inside

- `index.md` - the table of contents (start here)
- `companies/` - developers, OEMs, and marine contractors
- `projects/` - notable offshore wind farms
- `technology/` - cross-cutting technical concepts
- `policy/` - regulatory and permitting frameworks by region
- `digests/` - dated summaries of industry news, auto-generated by the ingestion pipeline
- `okf-cli.py` - the dependency-free navigation/search CLI
- `pipeline/` - the AgentMail-based ingestion pipeline that keeps this bundle current (see `pipeline/README.md`)
- `log.md` - change history

Primary source for news ingestion: [offshoreWIND.biz](https://www.offshorewind.biz/).
